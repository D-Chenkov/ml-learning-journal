**Key concept to bank (lived twice):** a container runs a BUILT IMAGE - a frozen snapshot of the code at build time. Editing source files does NOT change the running container; you must rebuild the image (then load + bump tag + apply). Source and running container are decoupled; the image is the frozen bridge, and only a rebuild refreshes it. openapi.json is the fast way to see which build is actually live.
